# NEM Account Info Checker

Check the Info of your account with NEM Address.

## Install and Run
``` bash
npm install
```
``` bash
ionic cordova run android
```
``` bash
ionic cordova run ios
```

## What info specifically:

1.  Status
    * Remote Status
    * Status
2.  Public Account
    * Address
    * Public Key
3.  Account
    * Balance
    * Vested Balance
    * Unvested Balance
4.  Harvesting
    * Harvested Blocks
    * Importance
5.  Cosigners Of (Array)
    * Address
    * Public Key
    * Balance
    * Vested Balance
    * Unvested Balance
6.  Cosigners (Array)
    * Address
    * Public Key
    * Balance
    * Vested Balance
    * Unvested Balance
